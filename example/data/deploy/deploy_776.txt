A list of hosts is displayed. The host list is retrieved from Active Directory and includes all computers that have a machine account. Not all machines in the list may be available for deployment because some of them may not be powered on or have been removed. If the host you want to deploy to is not in the list , a manual installation option is available by clicking the Manual Install button. See Manual Installation of Computer Recording Agent. Select the hosts that you would like to deploy the Computer Recording Agent to .
