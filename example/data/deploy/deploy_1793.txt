Bundle SQL Deploy when you distribute a new version of your application to clients and all your database changes will happen automatically. Simply save database changes from Enterprise Manager , SQL Analyser or Visual Studio .NET as scripts ( as most developers already do ) and this utility enables automatic data version upgrades. Designed for Winforms , Webforms , VB , ASP and Access applications ( and more ) , SSW SQL Deploy means you never have to worry about your Clients data when deploying applications again !
