PhpDock enables you to deploy any PHP web application as a Stand Alone Windows Desktop application w/o any changes in the code. PhpDock is a Web Server and Web Browser combined in one , small executable , capable of both running PHP scripts and rendering resulting HTML in the application Window. With PhpDock you can deploy the same PHP or HTML application both on the central Web Server and on each client as a Desktop application. PhpDock combines NuSphere 's powerful embeded Srv webserver ...
