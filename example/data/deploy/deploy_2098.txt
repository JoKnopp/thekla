ClientProtector ( SSCProtector ) server : SSCProt.dll. If you are not using IronWrapping your installer package must register the SSCProt.dll with the Windows system , and ( as usual ) it can deployed in the directory where you would normally place shared components or dlls. ( See your installers documentation for instructions how to register dlls ) . We recommend that you place the DLL in the System directory and that your installer marks it with Windows as " shared " .
