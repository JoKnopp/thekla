[ 0032 ] The plan script can include calls to one or more deployment commands configured to deploy software components to target computing devices and/or infrastructure resources. As such , the plan script can be generated in a procedural language. In other words , the plan script can be generated to include function calls that perform actions such as installing , uninstalling , updating , etc. In one embodiment , the computer language utilized as the plan script language can be any known , or to be known , scripting language. In another embodiment , the computer language utilized as the plan script language can be any known , or to be known , programming language. In another embodiment , the computer language utilized can be a scripting language exclusively known to the plan generator 202 and a plan execution engine 204 .
