These benefits are not limited to the Windows Server 2003 family. Microsoft Windows 2000 includes the Windows Installer technology. Versions of Windows Installer are also available for Microsoft Windows NT version 4.0 , Microsoft Windows 98 , and Microsoft Windows 95. However , using the Windows Server 2003 family or Windows 2000 does give administrators one major advantage : it combines the Windows Installer package files with the software installation and maintenance technology , so that administrators can easily deploy and manage software throughout their entire organization .
